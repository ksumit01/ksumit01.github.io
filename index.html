Inventory Management & ERP Integration

Manufacturing / Supply-Chain client
Dynamics 365Power PagesAzure FunctionsWorkday SOAP
Problem

Asset and inventory data lived in Workday ERP, disconnected from day-to-day operations, forcing manual lookups and error-prone re-entry.

Approach

Designed a Dataverse data model mirroring key ERP entities and an Azure Function SOAP client to synchronize records in near real time.

Solution

Built a Model-Driven App for internal asset tracking and a Power Pages portal giving external users self-service visibility, secured by role-based access.

Result

Real-time ERP visibility inside CRM, eliminating manual re-entry and giving operations a single source of truth for assets.

Enterprise CRM & Multi-System Integration

Enterprise SaaS client
Dynamics 365 CEC#JavaScriptPower AutomateAzure
Problem

Sales and service teams worked across several disconnected systems, with no unified customer view and heavy manual coordination.

Approach

Customized Dynamics 365 CE with C# plugins and JavaScript business logic, and orchestrated cross-system data movement through an Azure integration layer.

Solution

Secure Payment Integration

Services / Fintech client
Dynamics 365Stripe APIsAzure FunctionsC#
Problem

Payments were processed outside CRM, leaving orders and invoices out of sync and requiring manual reconciliation.

Approach

Implemented a Stripe PaymentIntent flow via Azure Functions, with a secure webhook receiver to capture payment events.

Solution

Automated the full order-to-invoice lifecycle inside Dynamics 365, with webhook-driven status updates writing back to CRM records.

Result

End-to-end payment visibility in CRM and elimination of manual invoice reconciliation.
